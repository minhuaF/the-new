### 尺寸要求
- **推荐尺寸**: 1080 x 1440 px (3:4 比例)
- **分辨率**: 高清，确保清晰度

## 🎨 文章分享卡片设计

### ArticleShareTemplate - 文章阅读笔记卡片

**尺寸**: 1080 x 1440 px

**布局结构**:
```
┌─────────────────────────┐
│  READING NOTES 标签      │
├──────────────┬──────────┤
│              │          │
│  左侧：       │  右侧：   │
│  文章内容     │  单词笔记 │
│              │  列表     │
│  • 标题      │  (最多8个)│
│  • 摘要      │          │
│  • 统计      │          │
│              │          │
├──────────────┴──────────┤
│  品牌信息 + 二维码        │
└─────────────────────────┘
```

**设计特点**:

**左侧文章区**:
- 📘 **文章标题**: 大字号 serif 字体，最多3行
- 📝 **文章摘要**: 前500字，最多12行
- 📊 **统计信息**: 已读文章数、标注单词数
- 🎨 **毛玻璃卡片**: 背景模糊，白色半透明

**右侧单词区**:
- 🏷️ **区域标题**: "My Words" 小卡片
- 📝 **单词列表**: 最多显示8个单词
- 每个单词卡片包含:
  - 单词 (serif 字体)
  - 音标 (彩色 mono 字体)
  - 中文释义 (简洁，最多2行)
- 💡 **省略提示**: 超过8个显示 "还有 X 个单词..."

**通用元素**:
- 🎯 **顶部标签**: "READING NOTES" 大写
- 🎨 **装饰背景**: 2个柔和模糊圆形
- 📲 **底部品牌**: Logo + 文案 + 二维码

**主题色方案**:
- Blue: rose-amber-sky 渐变
- Orange: orange-amber-yellow 渐变
- Green: emerald-green-teal 渐变
- Purple: purple-pink-rose 渐变
- Gray: slate-gray-zinc 渐变

## 🎨 设计规范

### 配色方案
```css
/* 背景渐变 */
rose-50 via amber-50 to sky-50

/* 主题色变体 */
- Blue: rose/amber/sky
- Orange: orange/amber/yellow
- Green: emerald/green/teal
- Purple: purple/pink/rose
- Gray: slate/gray/zinc
```

### 字体系统
```css
/* 主标题 */
font-family: 'Georgia, serif'
font-weight: 300 (font-light)
font-size: 120px (单词) / 60px (总结标题)

/* 小标签 */
text-transform: uppercase
letter-spacing: 0.2em
font-size: 24px
font-weight: 300

/* 正文 */
font-family: system-ui
font-weight: 300
font-size: 40px (释义) / 24px (正文)
```

### 圆角系统
```css
rounded-[48px]  /* 主卡片 */
rounded-3xl     /* 次要卡片、品牌卡片 */
rounded-2xl     /* 标签、二维码容器 */
rounded-full    /* 小标签、装饰圆形 */
```

### 毛玻璃效果
```css
bg-white/90 backdrop-blur-xl     /* 主卡片 */
bg-white/90 backdrop-blur-sm     /* 次要元素 */
border border-white/60           /* 边框 */
```

### 阴影系统
```css
shadow-2xl  /* 主卡片 */
shadow-xl   /* 次要卡片 */
shadow-lg   /* 品牌元素 */
```

## 🚀 使用方法

### 1. 在阅读页面点击分享

在文章阅读页面 (`/articles/[id]`)：
1. 阅读文章并标注单词
2. 点击右上角"分享"按钮
3. 打开分享对话框

### 2. 自定义分享卡片

在 ShareDialog 中：
- 选择主题色（5种配色方案）
- 实时预览效果
- 点击 "下载图片" 生成 PNG

### 3. 数据来源

分享卡片自动包含：
- **文章标题**: 自动获取
- **文章摘要**: 前500字
- **单词笔记**: 最多8个（按标注顺序）
  - 单词
  - 音标
  - 中文释义（取第一个释义）
- **统计信息**: 标注单词总数

### 4. 分享到小红书

1. 下载生成的 1080x1440 PNG 图片
2. 打开小红书 APP
3. 上传图片
4. 添加相关话题: #英语学习 #阅读笔记 #单词打卡
5. 发布！

## 💡 设计亮点

1. **3:4 黄金比例**: 完美适配小红书手机展示
2. **左右分栏布局**: 文章内容 + 单词笔记，信息密度高
3. **Modern Editorial 风格**: 杂志级排版，专业优雅
4. **毛玻璃层次**: 现代化视觉效果，层次分明
5. **柔和配色**: 温暖舒适，符合学习主题
6. **清晰易读**: 大字号 + 高对比度，一目了然
7. **自动化生成**: 无需手动输入，直接从文章数据生成
8. **品牌一致性**: 统一设计语言，识别度高
